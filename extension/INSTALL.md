# Installing the Terabox Downloader Extension

## Quick Start (5 minutes)

### Step 1: Open Chrome Extensions Page
1. Open Google Chrome
2. Type `chrome://extensions/` in the address bar
3. Press Enter

### Step 2: Enable Developer Mode
1. Look for "Developer mode" toggle in the top-right corner
2. Click to enable it
3. You'll see new buttons appear

### Step 3: Load the Extension
1. Click "Load unpacked" button
2. Navigate to your project folder
3. Select the `extension` folder
4. Click "Select Folder"

### Step 4: Verify Installation
1. You should see "Terabox Downloader Extension" in your extensions list

## Troubleshooting

### Extension not showing up?
- Make sure you selected the `extension` folder, not the parent folder
- Check that Developer mode is enabled
- Try reloading the extension (click refresh icon)

### "Extension not detected" message on web app?
- Refresh the web app page
- Make sure extension is enabled (toggle should be ON)
- Check browser console (F12) for errors

### Still getting verify_v2?
- Visit terabox.app first and make sure you're logged in
- Click the extension icon and click "Test Extension"
- Should show "âœ… Ready! X Terabox cookies detected"
- If no cookies, login to Terabox first

### Download link not working?
- Make sure you're using fresh cookies (visit Terabox recently)
- Try logging out and back in to Terabox
- Check if Terabox link is still valid
